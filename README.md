opPostalCodePlugin
==================

郵便番号を扱うプラグイン

下記 URL から取得しています

http://www.post.japanpost.jp/zipcode/dl/kogaki-zip.html


下記コマンドでデータのインサートができる（予定）
```
$ cd /path/to/openpne
$ php plugins/opPostalCodePlugin/bin/create_data.php
```

## 今後の予定

ajaxzip3 を使って入力補完ができるコンポーネントを作る
https://code.google.com/p/ajaxzip3/
